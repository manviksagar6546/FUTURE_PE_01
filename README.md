# AI Website Copy Generator for Local Businesses

## Internship
Prompt Engineering Internship – Future Interns

## Business Chosen
Brew Haven Café (Example Local Café)

## Objective
This project demonstrates how prompt engineering can be used to generate high-quality website copy for local businesses.

## Tools Used
- ChatGPT
- Claude
- Google Gemini
- GitHub

## Files Included
- homepage-copy.md → Homepage website copy
- services-page.md → Menu and services content
- cta-sections.md → Call-to-action sections
- prompts-used.md → Prompts used to generate content

## Outcome
Using structured prompts, AI can generate clear, persuasive website copy that local businesses can use for their websites.
